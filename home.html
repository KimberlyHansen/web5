GEOM99 WEB GIS DEVELOPMENT

View weather conditions at hiking trail locations in Ontario

Our Problem Statement

Identify your ideal state or outcome from completing the work

Users can view a live weather map and see Ontario hiking trail locations on the same map. The user can therefore more easily make decisions on where to go if they're planning on hiking

Document the reality of how things are done/ managed now

Live weather maps (precipitation, temperature, etc.) and hiking trail locations in Ontario are available but not within the same map

What are the consequences if your solution is not made/ nothing is changed?

Since weather maps and hiking trails in Ontario are not in the same location the user's experience is more cumbersome when planning their excursion.

Clearly state the proposal solutions in the context of the end user

A web map getting data from OpenWeatherMap API to be displayed with Ontario hiking trail locations.
